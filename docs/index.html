Features

Everything you need for Salesforce translations

From raw STF import to Salesforce-ready export — fully guided, validated, and auditable.

⚡

Auto-Translation

Google Translate (free), DeepL, Azure, or OpenAI. Translation Memory caches results so repeated strings never hit the API twice.

🛡

Smart Protection

Salesforce IDs, {!Placeholders}, MessageFormat tokens, HTML tags, URLs, and emails are never modified by the translator.

✅

Validation & Auto-Fix

Detects length overflows, missing placeholders, duplicate keys, and HTML mismatches. One-click auto-fix for most issues.

💾

Translation Memory

SQLite cache means the same string is never translated twice across runs. Saves time and API quota significantly.

📖

Glossary Support

Protect brand names from translation and enforce specific terms with a simple CSV glossary file.

🔄

Phase Snapshots

Reset any phase and restore cleanly from the previous phase's snapshot. No data loss, no stale state.

📊

Excel Audit Sheets

Every translated workbook includes Translation_Summary and Translation_Status_Log sheets for full auditability.

🔍

Validation Report

Export issues as a multi-sheet Excel workbook — one sheet per category (length_limit, duplicate_key, translation_failed, etc.).

🎯

Independent Phases

Jump into any phase directly with a Load button. No need to run the full pipeline — each phase works standalone.

Quick Start

Up and running in minutes

Download, install, and start translating. No Python, no setup, no command line.

1

Download the installer

Click the Download button above. The installer is ~47 MB and works on Windows 10 or later.

2

Run the setup wizard

Double-click the .exe and follow the wizard: Next → Next → Install. Creates a desktop shortcut automatically.

3

Import your STF file

Launch from the desktop shortcut. In Phase 1, click Browse and pick the .stf file you exported from Salesforce Translation Workbench.

4

Click Continue through each phase

The app guides you. Phase 2 converts to Excel, Phase 3 auto-translates, Phase 4 lets you review, Phase 5 validates, Phase 6 exports.

5

Upload the STF back to Salesforce

Phase 6 produces three STF files: Bilingual, Translated-only, and Untranslated-only. Upload Bilingual to Translation Workbench.

Requirements

What you need

The standalone installer bundles everything. No Python or dependencies needed.

⚠ System Requirements

  • Windows 10 or later (64-bit)
  • ~200 MB free disk space
  • Internet connection for translation API calls (Google Translate free tier, or paid backend)
  • No Python installation needed (fully standalone .exe)
