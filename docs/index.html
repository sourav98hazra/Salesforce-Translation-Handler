Salesforce Translation Manager

Professional desktop app for managing Salesforce Translation Workbench files. Six-phase pipeline: Import, Convert, Translate, Review, Validate, Export.

Key Features

Auto-Translation

Google Translate (free), DeepL, Azure, or OpenAI. Translation Memory caches results across runs.

Smart Protection

Salesforce IDs, placeholders, HTML tags, URLs, and emails are never modified by the translator.

Validation & Auto-Fix

Detects length overflows, missing placeholders, duplicate keys, and HTML mismatches. One-click fix.

Translation Memory

SQLite cache means the same string is never translated twice. Saves time and API quota.

Glossary Support

Protect brand names and enforce specific translations with a simple CSV glossary file.

Phase Snapshots

Reset any phase and restore cleanly. Each phase saves a snapshot for reliable undo.

Six-Phase Pipeline

  1. Import STF - Load the .stf file exported from Salesforce Translation Workbench
  2. STF to Excel - Convert to an organised workbook (one sheet per component type)
  3. Translate - Auto-translate untranslated rows with configurable backend
  4. Browse & Review - Edit translations inline, filter, find & replace
  5. Validate & Fix - Detect and auto-fix issues before import
  6. Export STF - Write the final .stf files ready for Salesforce upload

System Requirements

  • Windows 10 or later (64-bit)
  • No Python installation needed (standalone .exe)
  • Internet connection for translation API calls
  • ~200 MB disk space

Quick Start

  1. Download the installer from the button above
  2. Run the installer (Next > Next > Install)
  3. Launch from the desktop shortcut or Start Menu
  4. Import your .stf file in Phase 1
  5. Click Continue through each phase - the app guides you
  6. Export the final .stf files in Phase 6 and upload to Salesforce
